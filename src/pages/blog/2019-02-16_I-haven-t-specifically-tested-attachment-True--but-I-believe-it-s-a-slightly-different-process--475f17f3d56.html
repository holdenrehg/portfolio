I haven’t specifically tested attachment=True, but I believe it’s a slightly different process. You would probably need to search ir.attachment to find the proper attachment associated with the record and field, then update the binary string there.

For website optimization, what you are doing already is the only thing I’m aware of. Odoo does not handle the optimization of images, like you said. Honestly I’m not sure you would want them to do that automatically, it could be unexpected in certain situations.

So yes, you’ll have to use something like PIL to format the images before inserting them into the e-commerce images field on products.
